## Документ 1: Акт выполненных работ (AktVypolnennyhRabot)

### 1.1 Создание объекта

1. Дерево конфигурации → **Документы** → правый клик → **Добавить**
2. Вкладка **Основные**:

| Поле | Значение |
|---|---|
| Имя | AktVypolnennyhRabot |
| Синоним | Акт выполненных работ |
| Комментарий | Первичный документ по выполненным работам на котельной |

3. Вкладка **Нумерация**:

| Поле | Значение |
|---|---|
| Длина номера | 9 |
| Тип номера | Строка (выбрать радиокнопку) |
| Периодичность | Непериодический (или Год — по желанию) |
| Контроль уникальности | поставить флажок |

> **Префикс в GUI не задаётся.** Поля «Префикс» на вкладке Нумерация нет. Префикс «АКТ-» добавляется одним из двух способов:
> - **Способ A (рекомендую):** оставить номер без префикса — он будет просто «000000001». Для учебного проекта этого достаточно.
> - **Способ B:** в модуле объекта добавить в процедуру `ОбработкаЗаписи` строку: `Если ПустаяСтрока(Номер) Тогда Номер = "АКТ-" + Формат(ТекущаяДата(), "ДФ=yyyyMMdd"); КонецЕсли;` — но это усложняет конфигурацию. Для Дня 14 используем Способ A.

### 1.2 Реквизиты (вкладка «Данные»)

Перейти на вкладку **Данные** → в дереве выделить **Реквизиты** → добавлять кнопкой «+».

Добавить реквизиты по одному:

| Имя | Синоним | Тип |
|---|---|---|
| Котельная | Котельная | СправочникСсылка.Kotelnye |
| Заказчик | Заказчик | СправочникСсылка.Kontragenty |
| БригадаНаименование | Бригада | Строка, длина 100 |
| ДатаНачалаРабот | Дата начала работ | Дата |
| ДатаОкончанияРабот | Дата окончания работ | Дата |
| СуммаУслуг | Сумма услуг | Число, длина 15, точность 2 |
| СуммаМатериалов | Сумма материалов | Число, длина 15, точность 2 |
| СуммаИтого | Сумма итого | Число, длина 15, точность 2 |
| НДС | НДС | Число, длина 15, точность 2 |
| ВсегоКОплате | Всего к оплате | Число, длина 15, точность 2 |
| СтатьяЗатрат | Статья затрат | СправочникСсылка.StatiZatrat |
| Подразделение | Подразделение | СправочникСсылка.Subdivisions |

### 1.3 Табличная часть «Материалы»

1. На вкладке **Данные** → выделить узел **Табличные части** → нажать **«+»** на верхней панели инструментов
2. Справа откроется панель свойств — в поле **Имя** написать `Материалы`, Синоним заполнится автоматически
3. Теперь нужно добавить колонки: в дереве **раскрыть** узел **Материалы** (нажать стрелку ▶ слева) — появится вложенный узел **Реквизиты**
4. Выделить **Реквизиты** внутри Материалы → нажать **«+»** → добавить колонки по одной:

| Имя | Синоним | Тип |
|---|---|---|
| Номенклатура | Номенклатура | СправочникСсылка.Nomenklatura |
| Количество | Количество | Число, длина 15, точность 3 |
| Цена | Цена | Число, длина 15, точность 2 |
| Сумма | Сумма | Число, длина 15, точность 2 |

> Важно: «+» нажимать при выделенном узле **Реквизиты внутри Материалы**, а не верхнем узле «Реквизиты» документа — иначе колонка добавится к реквизитам документа, а не к табличной части.

### 1.4 Регистрация движений (вкладка «Движения»)

1. Перейти на вкладку **Движения**
2. В списке регистров найти и поставить флажки:
   - [x] **БухгалтерскийОсновной** (Регистры бухгалтерии)
   - [x] **МатериалыНаСкладе** (Регистры накопления)
3. **Ctrl+S**

### 1.5 Модуль объекта

1. Правый клик на **AktVypolnennyhRabot** в дереве → **Модуль объекта**
2. Откроется редактор кода — вставить процедуру целиком:

```1c
Процедура ОбработкаПроведения(Отказ, РежимПроведения)
	
	// ── Счета из плана счетов ──────────────────────────────────────────────
	Сч20     = ПланыСчетов.Основной.НайтиПоКоду("20");
	Сч62     = ПланыСчетов.Основной.НайтиПоКоду("62");
	Сч9001   = ПланыСчетов.Основной.НайтиПоКоду("90.01");
	Сч9002   = ПланыСчетов.Основной.НайтиПоКоду("90.02");
	СчётКт10 = ПланыСчетов.Основной.НайтиПоКоду("10.01");
	
	// ── 1. Списание материалов со склада (регистр накопления) ──────────────
	Движения.МатериалыНаСкладе.Записывать = Истина;
	
	Для Каждого СтрокаТЧ Из Материалы Цикл
		
		Если СтрокаТЧ.Количество = 0 Тогда
			Продолжить;
		КонецЕсли;
		
		Движение = Движения.МатериалыНаСкладе.Добавить();
		Движение.ВидДвижения      = ВидДвиженияНакопления.Расход;
		Движение.Период           = Дата;
		Движение.Номенклатура     = СтрокаТЧ.Номенклатура;
		Движение.Количество       = СтрокаТЧ.Количество;
		Движение.СтоимостьВРублях = СтрокаТЧ.Сумма;
		
	КонецЦикла;
	
	// ── 2. Проводки в регистр бухгалтерии ─────────────────────────────────
	Движения.БухгалтерскийОсновной.Записывать = Истина;
	
	// 2.1 Дт 20 Кт 10.01 — списание материалов на производство
	Для Каждого СтрокаТЧ Из Материалы Цикл
		
		Если СтрокаТЧ.Сумма = 0 Тогда
			Продолжить;
		КонецЕсли;
		
		Проводка = Движения.БухгалтерскийОсновной.Добавить();
		Проводка.Период      = Дата;
		Проводка.СчетДт      = Сч20;
		Проводка.СчетКт      = СчётКт10;
		Проводка.Сумма       = СтрокаТЧ.Сумма;
		Проводка.Количество  = СтрокаТЧ.Количество;
		Проводка.Содержание  = "Списание: " + СтрокаТЧ.Номенклатура.Наименование;
		
	КонецЦикла;
	
	// 2.2 Дт 62 Кт 90.01 — реализация работ заказчику
	Если ВсегоКОплате <> 0 Тогда
		
		Проводка = Движения.БухгалтерскийОсновной.Добавить();
		Проводка.Период     = Дата;
		Проводка.СчетДт     = Сч62;
		Проводка.СчетКт     = Сч9001;
		Проводка.Сумма      = ВсегоКОплате;
		Проводка.Содержание = "Реализация работ: " + Котельная.Наименование;
		
	КонецЕсли;
	
	// 2.3 Дт 90.02 Кт 20 — списание себестоимости
	Если СуммаМатериалов <> 0 Тогда
		
		Проводка = Движения.БухгалтерскийОсновной.Добавить();
		Проводка.Период     = Дата;
		Проводка.СчетДт     = Сч9002;
		Проводка.СчетКт     = Сч20;
		Проводка.Сумма      = СуммаМатериалов;
		Проводка.Содержание = "Себестоимость работ по акту " + Номер;
		
	КонецЕсли;
	
КонецПроцедуры
```

3. **Ctrl+S**

### 1.6 Форма документа

#### Создание формы

1. В дереве конфигурации раскрыть **AktVypolnennyhRabot** → выделить узел **Формы**
2. Правый клик на **Формы** → **Добавить**
3. Откроется **Конструктор формы документа** (как на скриншоте)
4. Убедиться что выбрана радиокнопка **Форма документа**
5. Флажок **«Назначить форму основной»** — оставить включённым
6. Имя: `ФормаДокумента` (заполнится автоматически)
7. Нажать **Готово** (не «Далее», сразу «Готово»)
8. Откроется визуальный редактор формы со стандартными полями

#### Добавление кнопки «Рассчитать суммы»

Порядок важен: сначала создать команду, потом написать код, потом привязать кнопку.

**Шаг 1 — Создать команду формы**

1. В редакторе формы перейти на правую панель → вкладка **Команды** → подвкладка **Команды формы**
2. Нажать **«+»** на панели инструментов над списком
3. Появится новая строка — написать имя: `РассчитатьСуммы`
4. Синоним заполнится автоматически: «Рассчитать суммы»

**Шаг 2 — Написать код в модуле**

1. Нажать вкладку **Модуль** внизу редактора формы
2. Вставить процедуру:

```1c
&НаКлиенте
Процедура РассчитатьСуммы(Команда)
	
	СуммаМат = 0;
	Для Каждого СтрокаТЧ Из Объект.Материалы Цикл
		СтрокаТЧ.Сумма = СтрокаТЧ.Количество * СтрокаТЧ.Цена;
		СуммаМат = СуммаМат + СтрокаТЧ.Сумма;
	КонецЦикла;
	
	Объект.СуммаМатериалов = СуммаМат;
	Объект.СуммаИтого      = Объект.СуммаУслуг + СуммаМат;
	Объект.НДС             = Объект.СуммаИтого * 20 / 120;
	Объект.ВсегоКОплате    = Объект.СуммаИтого;
	
КонецПроцедуры
```

**Шаг 3 — Привязать команду к кнопке**

1. Вернуться на вкладку **Форма** внизу редактора
2. В дереве элементов (левая панель) выделить **КнопкаРассчитать**
3. В панели **Свойства** (правая сторона экрана) найти поле **ИмяКоманды**
4. Нажать кнопку **«...»** справа от поля
5. В открывшемся списке выбрать **РассчитатьСуммы** (команда появится, так как мы создали её на Шаге 1)
6. **Ctrl+S**

> Если кнопки КнопкаРассчитать ещё нет в дереве элементов: правый клик на корневом элементе дерева → **Добавить** → **Кнопка** → в свойствах задать Имя `КнопкаРассчитать`, Заголовок `Рассчитать суммы`, затем привязать ИмяКоманды как описано выше.

### 4.6 Модуль объекта

Правый клик → **Модуль объекта** → вставить:

```1c
Процедура ОбработкаПроведения(Отказ, РежимПроведения)
	
	// ── Счета из плана счетов ──────────────────────────────────────────────
	Сч20   = ПланыСчетов.Основной.НайтиПоКоду("20");
	Сч70   = ПланыСчетов.Основной.НайтиПоКоду("70");
	Сч6801 = ПланыСчетов.Основной.НайтиПоКоду("68.01");
	
	// ── 1. Начисления: проводки и регистр накопления ──────────────────────
	Движения.БухгалтерскийОсновной.Записывать = Истина;
	Движения.НачисленияСотрудникам.Записывать = Истина;
	
	Для Каждого СтрокаНач Из Начисления Цикл
		
		Если СтрокаНач.Сумма = 0 Тогда
			Продолжить;
		КонецЕсли;
		
		// Проводка Дт 20 Кт 70 (начисление зарплаты)
		Проводка = Движения.БухгалтерскийОсновной.Добавить();
		Проводка.Период     = Дата;
		Проводка.СчетДт     = Сч20;
		Проводка.СчетКт     = Сч70;
		Проводка.Сумма      = СтрокаНач.Сумма;
		Проводка.Содержание = "Начисление: " + СтрокаНач.ВидНачисления.Наименование
		                      + " / " + СтрокаНач.Сотрудник.Наименование;
		
		// Запись в регистр накопления НачисленияСотрудникам
		ДвижНач = Движения.НачисленияСотрудникам.Добавить();
		ДвижНач.ВидДвижения     = ВидДвиженияНакопления.Приход;
		ДвижНач.Период          = Дата;
		ДвижНач.Сотрудник       = СтрокаНач.Сотрудник;
		ДвижНач.ВидНачисления   = СтрокаНач.ВидНачисления;
		ДвижНач.РасчетныйПериод = МесяцПериода;
		ДвижНач.СуммаНачисления = СтрокаНач.Сумма;
		ДвижНач.СуммаУдержания  = 0;
		ДвижНач.СуммаКВыплате   = СтрокаНач.Сумма;
		
	КонецЦикла;
	
	// ── 2. Удержания: проводки ────────────────────────────────────────────
	Для Каждого СтрокаУд Из Удержания Цикл
		
		Если СтрокаУд.Сумма = 0 Тогда
			Продолжить;
		КонецЕсли;
		
		// Проводка Дт 70 Кт 68.01 (удержание НДФЛ)
		Проводка = Движения.БухгалтерскийОсновной.Добавить();
		Проводка.Период     = Дата;
		Проводка.СчетДт     = Сч70;
		Проводка.СчетКт     = Сч6801;
		Проводка.Сумма      = СтрокаУд.Сумма;
		Проводка.Содержание = "Удержание: " + СтрокаУд.ВидУдержания.Наименование
		                      + " / " + СтрокаУд.Сотрудник.Наименование;
		
		// Запись в регистр накопления — расход (удержание уменьшает выплату)
		ДвижУд = Движения.НачисленияСотрудникам.Добавить();
		ДвижУд.ВидДвижения     = ВидДвиженияНакопления.Расход;
		ДвижУд.Период          = Дата;
		ДвижУд.Сотрудник       = СтрокаУд.Сотрудник;
		ДвижУд.ВидНачисления   = СтрокаУд.ВидУдержания;
		ДвижУд.РасчетныйПериод = МесяцПериода;
		ДвижУд.СуммаНачисления = 0;
		ДвижУд.СуммаУдержания  = СтрокаУд.Сумма;
		ДвижУд.СуммаКВыплате   = -СтрокаУд.Сумма;
		
	КонецЦикла;
	
КонецПроцедуры
```

## Возможные ошибки при проведении

| Ошибка | Причина | Решение |
|---|---|---|
| «Поле объекта не обнаружено: СчётУчёта» | Реквизит Номенклатуры не переименован или не обновлена БД | Проверить что реквизит Номенклатура.СчётУчёта существует и тип ПланСчетовСсылка.Osnovnoy |
| «Поле объекта не обнаружено: СчётКредита» | Реквизит ВидыНачислений не добавлен | Открыть справочник VidyNachisleniy, проверить реквизит СчётКредита |
| «Недостаточно остатков» | В регистре MaterialyNaSklade нет прихода материалов | Сначала создать документ поступления материалов (День 15) или добавить приход вручную |
| «Поле Организация не заполнено» | Реквизит Организация в регистре бухгалтерии обязателен | В коде проверить `Проводка.Организация = "Основная"` — строка должна совпадать с реальным значением |
| «Обработчик ОбработкаПроведения не найден» | Модуль объекта пустой или процедура не сохранена | Открыть модуль объекта, убедиться что процедура есть, Ctrl+S |
