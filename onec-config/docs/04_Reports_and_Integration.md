## Часть 1: Отчёт «Оборотно-сальдовая ведомость»

### 1.1 Создание объекта

1. Дерево конфигурации → **Отчёты** → правый клик → **Добавить**
2. Вкладка **Основные**:

| Поле | Значение |
|---|---|
| Имя | OborotnoSaldovayaVedomost |
| Синоним | Оборотно-сальдовая ведомость |

3. На вкладке **Основные** нажать кнопку **«Открыть схему компоновки данных»**
4. Появится диалог — выбрать **«Создать»** → **OK**
5. Откроется редактор схемы компоновки данных (СКД)

### 1.2 Набор данных

1. В левой панели СКД выделить вкладку **«Наборы данных»**
2. Нажать кнопку **«+»** на панели инструментов → выбрать **«Добавить набор данных — Запрос»**
3. В поле **«Запрос»** нажать кнопку **«...»** (или «Конструктор запроса») и вставить запрос вручную:

```sql
ВЫБРАТЬ
    ОстаткиИОбороты.Счет.Код КАК КодСчета,
    ОстаткиИОбороты.Счет.Наименование КАК НаименованиеСчета,
    ОстаткиИОбороты.СуммаНачальныйОстатокДт КАК СальдоНачалоДт,
    ОстаткиИОбороты.СуммаНачальныйОстатокКт КАК СальдоНачалоКт,
    ОстаткиИОбороты.СуммаОборотДт КАК ОборотДт,
    ОстаткиИОбороты.СуммаОборотКт КАК ОборотКт,
    ОстаткиИОбороты.СуммаКонечныйОстатокДт КАК СальдоКонецДт,
    ОстаткиИОбороты.СуммаКонечныйОстатокКт КАК СальдоКонецКт
ИЗ
    РегистрБухгалтерии.БухгалтерскийОсновной.ОстаткиИОбороты(
        &ПериодС,
        &ПериодПо,
        Регистратор,
        ,
    ) КАК ОстаткиИОбороты
УПОРЯДОЧИТЬ ПО
    КодСчета
```

> **Важно:** имена регистра и плана счетов пишем кириллицей: `БухгалтерскийОсновной`, `Основной`. Параметр `Организация` убран из условия виртуальной таблицы — в учебной конфигурации он не заполнен в проводках.

4. Нажать **OK** для закрытия редактора запроса

### 1.3 Параметры

1. Перейти на вкладку **«Параметры»** в СКД
2. Параметры `ПериодС` и `ПериодПо` должны появиться автоматически из запроса
3. Для каждого установить:
   - Тип: **Дата**
   - Заголовок: «Период с» / «Период по»
   - Флажок **«Включать в доступные поля»** — снять (параметры не нужны как колонки)

### 1.4 Настройки СКД

Перейти на вкладку **«Настро...»** (крайняя правая вкладка в СКД).

В правой части видно дерево: **Отчёт** → **«Детальные записи»** (уже создано автоматически).

**Настроить поля вывода:**

1. В дереве выделить **«Детальные записи»**
2. В нижней панели перейти на подвкладку **«Выбранные поля»**
3. Слева — колонка **«Доступные поля»**, справа — пустая колонка **«Выбранные поля»**
4. Двойным кликом по каждому полю из левой колонки перенести его в правую:
   - КодСчета
   - НаименованиеСчета
   - СальдоНачалоДт
   - СальдоНачалоКт
   - ОборотДт
   - ОборотКт
   - СальдоКонецДт
   - СальдоКонецКт

**Настроить параметры:**

1. В нижней панели перейти на подвкладку **«Пара...»** (Параметры)
2. Там уже видны **Начало периода** и **Конец периода** — они соответствуют параметрам `&ПериодС` и `&ПериодПо` из запроса
3. Поставить флажки слева у обоих параметров — чтобы они отображались в форме отчёта при запуске

**Ctrl+S** в окне СКД.

### 1.5 Сохранение

1. **Ctrl+S** в редакторе СКД
2. Закрыть окно СКД
3. **Ctrl+S** в главном окне конфигуратора

## Часть 2: Отчёт «Карточка счёта»

2. **«Открыть схему компоновки данных»** → **«Создать»**

### 2.2 Набор данных — запрос

```sql
ВЫБРАТЬ
    Движения.Период КАК Период,
    Движения.Регистратор.Представление КАК Документ,
    Движения.СчетДт.Код КАК СчетДт,
    Движения.СчетКт.Код КАК СчетКт,
    Движения.Сумма КАК Сумма,
    Движения.Содержание КАК Содержание
ИЗ

ГДЕ

    И (Движения.СчетДт = &Счет
        ИЛИ Движения.СчетКт = &Счет)
УПОРЯДОЧИТЬ ПО
    Движения.Период
```

### 2.3 Параметры

| Имя | Тип |
|---|---|
| ПериодС | Дата |
| ПериодПо | Дата |
| Счет | ПланСчетовСсылка.Основной |

Поставить флажки у всех трёх параметров чтобы они отображались при запуске отчёта.

### 2.4 Настройки

Перейти на вкладку **«Настройки»**.

В правой части дерево: **Отчёт** → **«Детальные записи»** (создано автоматически).

**Настроить поля вывода:**

1. В дереве выделить **«Детальные записи»**

3. Слева — колонка **«Доступные поля»**, справа — **«Выбранные поля»**
4. Двойным кликом по каждому полю из левой колонки перенести его в правую:
   - Период
   - Документ
   - СчетДт
   - СчетКт
   - Сумма
   - Содержание

**Ctrl+S** в окне СКД → закрыть СКД → **Ctrl+S** в конфигураторе.

### 4.2 Набор данных — запрос

```sql
ВЫБРАТЬ
    Начисления.Сотрудник КАК Сотрудник,
    Начисления.Сотрудник.Наименование КАК ФИО,
    СУММА(Начисления.СуммаНачисления) КАК СуммаНачислений,
    СУММА(Начисления.СуммаУдержания) КАК СуммаУдержаний,
    СУММА(Начисления.СуммаКВыплате) КАК СуммаКВыплате

    РегистрНакопления.НачисленияСотрудникам КАК Начисления
```

## Часть 5: Обработка «Загрузка данных из XML»

### 5.1 Создание объекта

1. Дерево → **Обработки** → правый клик → **Добавить**

| Поле | Значение |
|---|---|
| Имя | ZagruzkaIzXML |
| Синоним | Загрузка данных из XML |

### 5.2 Реквизиты (вкладка «Данные»)

Вкладка **Данные** → выделить **Реквизиты** → «+» для каждого:

| Имя | Синоним | Тип |
|---|---|---|
| ПутьКФайлу | Путь к файлу | Строка, длина 500 |
| ТипДанных | Тип данных | Строка, длина 50 |

### 5.3 Форма обработки

1. Раскрыть **ZagruzkaIzXML** → выделить **Формы** → правый клик → **Добавить**
2. В конструкторе выбрать **Форма обработки** → **Готово**

**Добавить элементы формы вручную:**

В дереве элементов (левая панель редактора формы):
- Правый клик на корне → **Добавить** → **Поле ввода** → Имя `ПолеПути`, привязать к реквизиту `ПутьКФайлу` через свойство **ПутьКДанным** = `Объект.ПутьКФайлу`
- Правый клик → **Добавить** → **Поле ввода** → Имя `ПолеТипа`, привязать к `Объект.ТипДанных`
- Правый клик → **Добавить** → **Кнопка** → Имя `КнопкаЗагрузить`, Заголовок `Загрузить`

**Создать команду:**
Вкладка **Команды** → **Команды формы** → «+» → Имя `Загрузить`

### 5.4 Модуль формы

Вкладка **Модуль** → вставить:

```1c
&НаКлиенте
Процедура Загрузить(Команда)
	
	Если ПустаяСтрока(Объект.ПутьКФайлу) Тогда
		Сообщить("Укажите путь к XML-файлу");
		Возврат;
	КонецЕсли;
	
	Если ПустаяСтрока(Объект.ТипДанных) Тогда
		Сообщить("Укажите тип данных: Акты, Материалы или Табель");
		Возврат;
	КонецЕсли;
	
	Результат = ЗагрузитьНаСервере(Объект.ПутьКФайлу, Объект.ТипДанных);
	Сообщить("Загружено документов: " + Результат);
	
КонецПроцедуры

&НаСервере
Функция ЗагрузитьНаСервере(Путь, ТипДанных)
	
	Если Не ФайлСуществует(Путь) Тогда
		Сообщить("Файл не найден: " + Путь);
		Возврат 0;
	КонецЕсли;
	
	Чтение = Новый ЧтениеXML;
	Чтение.ОткрытьФайл(Путь);
	
	Счетчик = 0;
	ТекущийДокумент = Новый Структура;
	ВнутриДокумента = Ложь;
	ВнутриМатериалов = Ложь;
	ТекущийМатериал = Новый Структура;
	СписокМатериалов = Новый Массив;
	
	Пока Чтение.Прочитать() Цикл
		
		Если Чтение.ТипУзла = ТипУзлаXML.НачалоЭлемента Тогда
			
			Если Чтение.Имя = "Документ" Тогда
				ВнутриДокумента = Истина;
				ТекущийДокумент = Новый Структура("Номер, Дата, Котельная, Заказчик, БригадаНаименование, СуммаУслуг, СуммаМатериалов, ВсегоКОплате");
				СписокМатериалов = Новый Массив;
				
			ИначеЕсли Чтение.Имя = "Материалы" Тогда
				ВнутриМатериалов = Истина;
				
			ИначеЕсли Чтение.Имя = "Материал" И ВнутриМатериалов Тогда
				ТекущийМатериал = Новый Структура("Номенклатура, Количество, Цена, Сумма");
				
			ИначеЕсли ВнутриДокумента Тогда
				ИмяПоля = Чтение.Имя;
				Чтение.Прочитать();
				Если Чтение.ТипУзла = ТипУзлаXML.Текст Тогда
					Значение = Чтение.Значение;
					Если ВнутриМатериалов И ТекущийМатериал.Свойство(ИмяПоля) Тогда
						ТекущийМатериал[ИмяПоля] = Значение;
					ИначеЕсли ТекущийДокумент.Свойство(ИмяПоля) Тогда
						ТекущийДокумент[ИмяПоля] = Значение;
					КонецЕсли;
				КонецЕсли;
			КонецЕсли;
			
		ИначеЕсли Чтение.ТипУзла = ТипУзлаXML.КонецЭлемента Тогда
			
			Если Чтение.Имя = "Материал" И ВнутриМатериалов Тогда
				СписокМатериалов.Добавить(Новый Структура(
					"Номенклатура, Количество, Цена, Сумма",
					ТекущийМатериал.Номенклатура,
					?(ПустаяСтрока(ТекущийМатериал.Количество), 0, Число(ТекущийМатериал.Количество)),
					?(ПустаяСтрока(ТекущийМатериал.Цена),       0, Число(ТекущийМатериал.Цена)),
					?(ПустаяСтрока(ТекущийМатериал.Сумма),      0, Число(ТекущийМатериал.Сумма))
				));
				
			ИначеЕсли Чтение.Имя = "Материалы" Тогда
				ВнутриМатериалов = Ложь;
				
			ИначеЕсли Чтение.Имя = "Документ" Тогда
				ВнутриДокумента = Ложь;
				
				Если ТипДанных = "Акты" Тогда
					Если СоздатьАкт(ТекущийДокумент, СписокМатериалов) Тогда
						Счетчик = Счетчик + 1;
					КонецЕсли;
				ИначеЕсли ТипДанных = "Материалы" Тогда
					Если СоздатьСписание(ТекущийДокумент, СписокМатериалов) Тогда
						Счетчик = Счетчик + 1;
					КонецЕсли;
				ИначеЕсли ТипДанных = "Табель" Тогда
					Если СоздатьТабель(ТекущийДокумент, СписокМатериалов) Тогда
						Счетчик = Счетчик + 1;
					КонецЕсли;
				КонецЕсли;
				
			КонецЕсли;
			
		КонецЕсли;
```

- Поле ввода привязать к `Объект.ПериодС` (Имя `ПолеПериодС`)
- Поле ввода привязать к `Объект.ПериодПо` (Имя `ПолеПериодПо`)
- Поле ввода привязать к `Объект.ПутьСохранения` (Имя `ПолеПути`)

Создать `backend/app/services/onec_xml_export.py`:

```python
from __future__ import annotations

import os
from datetime import date, datetime
from decimal import Decimal
from pathlib import Path
from typing import Any
from xml.etree import ElementTree as ET
from xml.dom import minidom

from sqlalchemy import text
from sqlalchemy.ext.asyncio import AsyncSession


EXPORT_DIR = Path("reports/onec-export")


def _pretty_xml(root: ET.Element) -> str:
    raw = ET.tostring(root, encoding="unicode")
    return minidom.parseString(raw).toprettyxml(indent="    ", encoding=None)


class OneCXMLExporter:

    def __init__(self, session: AsyncSession):
        self._session = session
        EXPORT_DIR.mkdir(parents=True, exist_ok=True)

    async def export_acts_for_period(self, period: str) -> str:
        """period = 'YYYY-MM', e.g. '2026-04'"""
        year, month = int(period[:4]), int(period[5:7])
        date_from = date(year, month, 1)
        if month == 12:
            date_to = date(year + 1, 1, 1)
        else:
            date_to = date(year, month + 1, 1)

        rows = await self._session.execute(text("""
            SELECT
                w.number,
                w.created_at,
                b.name  AS boiler_name,
                c.name  AS customer_name,
                br.name AS brigade_name,
                w.id    AS work_order_id
            FROM work_orders w
            JOIN requests r   ON r.id = w.request_id
            JOIN boilers  b   ON b.id = r.boiler_id
            LEFT JOIN customers c   ON c.id = r.customer_id
            LEFT JOIN brigades  br  ON br.id = w.brigade_id
            WHERE w.created_at >= :df AND w.created_at < :dt
              AND w.status = 'completed'
        """), {"df": date_from, "dt": date_to})
        acts = rows.fetchall()

        root = ET.Element("Выгрузка", ТипДанных="Акты", Период=period)

        for act in acts:
            materials = await self._get_work_order_materials(act.work_order_id)
            doc = ET.SubElement(root, "Документ")
            ET.SubElement(doc, "Номер").text = str(act.number or "")
            ET.SubElement(doc, "Дата").text = (
                act.created_at.date().isoformat()
                if act.created_at else date_from.isoformat()
            )
            ET.SubElement(doc, "Котельная").text = act.boiler_name or ""
            ET.SubElement(doc, "Заказчик").text = act.customer_name or ""
            ET.SubElement(doc, "БригадаНаименование").text = act.brigade_name or ""

            total_mat = sum(Decimal(str(m["sum"])) for m in materials)
            ET.SubElement(doc, "СуммаУслуг").text = "0"
            ET.SubElement(doc, "СуммаМатериалов").text = str(total_mat)
            ET.SubElement(doc, "ВсегоКОплате").text = str(total_mat)

            mats_el = ET.SubElement(doc, "Материалы")
            for m in materials:
                mat = ET.SubElement(mats_el, "Материал")
                ET.SubElement(mat, "Номенклатура").text = m["name"]
                ET.SubElement(mat, "Количество").text = str(m["qty"])
                ET.SubElement(mat, "Цена").text = str(m["price"])
                ET.SubElement(mat, "Сумма").text = str(m["sum"])

        path = EXPORT_DIR / f"acts_{period}.xml"
        path.write_text(_pretty_xml(root), encoding="utf-8")
        return str(path)

    async def export_materials_for_period(self, period: str) -> str:
        year, month = int(period[:4]), int(period[5:7])
        date_from = date(year, month, 1)
        date_to = date(year, month + 1, 1) if month < 12 else date(year + 1, 1, 1)

        rows = await self._session.execute(text("""
            SELECT mm.id, mm.movement_date, b.name AS boiler_name,
                   mm.total_cost
            FROM material_movements mm
            LEFT JOIN boilers b ON b.id = mm.boiler_id
            WHERE mm.movement_date >= :df AND mm.movement_date < :dt
              AND mm.movement_type = 'writeoff'
        """), {"df": date_from, "dt": date_to})
        movements = rows.fetchall()

        root = ET.Element("Выгрузка", ТипДанных="Материалы", Период=period)
        for mv in movements:
            items = await self._get_movement_items(mv.id)
            doc = ET.SubElement(root, "Документ")
            ET.SubElement(doc, "Дата").text = mv.movement_date.isoformat()
            ET.SubElement(doc, "Котельная").text = mv.boiler_name or ""
            ET.SubElement(doc, "СуммаМатериалов").text = str(mv.total_cost or 0)
            mats_el = ET.SubElement(doc, "Материалы")
            for item in items:
                mat = ET.SubElement(mats_el, "Материал")
                ET.SubElement(mat, "Номенклатура").text = item["name"]
                ET.SubElement(mat, "Количество").text = str(item["qty"])
                ET.SubElement(mat, "Цена").text = str(item["price"])
                ET.SubElement(mat, "Сумма").text = str(item["sum"])

        path = EXPORT_DIR / f"materials_{period}.xml"
        path.write_text(_pretty_xml(root), encoding="utf-8")
        return str(path)

    async def export_timesheet_for_period(self, period: str) -> str:
        year, month = int(period[:4]), int(period[5:7])
        date_from = date(year, month, 1)
        date_to = date(year, month + 1, 1) if month < 12 else date(year + 1, 1, 1)

        rows = await self._session.execute(text("""
            SELECT ts.id, ts.work_date, e.full_name, ts.hours_worked,
                   d.name AS dept_name
            FROM timesheets ts
            JOIN employees e ON e.id = ts.employee_id
            LEFT JOIN departments d ON d.id = e.department_id
            WHERE ts.work_date >= :df AND ts.work_date < :dt
        """), {"df": date_from, "dt": date_to})
        rows_list = rows.fetchall()

        root = ET.Element("Выгрузка", ТипДанных="Табель", Период=period)
        doc = ET.SubElement(root, "Документ")
        ET.SubElement(doc, "Дата").text = date_from.isoformat()
        ET.SubElement(doc, "Котельная").text = ""
        mats_el = ET.SubElement(doc, "Материалы")
        for row in rows_list:
            mat = ET.SubElement(mats_el, "Материал")
            ET.SubElement(mat, "Номенклатура").text = row.full_name
            ET.SubElement(mat, "Количество").text = str(row.hours_worked or 0)
            ET.SubElement(mat, "Цена").text = "0"
            ET.SubElement(mat, "Сумма").text = "0"

        path = EXPORT_DIR / f"timesheet_{period}.xml"
        path.write_text(_pretty_xml(root), encoding="utf-8")
        return str(path)

    async def _get_work_order_materials(self, work_order_id: int) -> list[dict]:
        rows = await self._session.execute(text("""
            SELECT m.name, wom.quantity AS qty,
                   wom.unit_price AS price,
                   (wom.quantity * wom.unit_price) AS sum
            FROM work_order_materials wom
            JOIN materials m ON m.id = wom.material_id
            WHERE wom.work_order_id = :woid
        """), {"woid": work_order_id})
        return [dict(r._mapping) for r in rows.fetchall()]

    async def _get_movement_items(self, movement_id: int) -> list[dict]:
        rows = await self._session.execute(text("""
            SELECT m.name, mmi.quantity AS qty,
                   mmi.unit_price AS price,
                   (mmi.quantity * mmi.unit_price) AS sum
            FROM material_movement_items mmi
            JOIN materials m ON m.id = mmi.material_id
            WHERE mmi.movement_id = :mid
        """), {"mid": movement_id})
        return [dict(r._mapping) for r in rows.fetchall()]
```

### 7.2 Роутер интеграции

Создать `backend/app/routers/integration.py`:

```python
from fastapi import APIRouter, Depends
from sqlalchemy.ext.asyncio import AsyncSession

from app.database import get_session
from app.services.onec_xml_export import OneCXMLExporter

router = APIRouter(prefix="/integration", tags=["integration"])


@router.post("/onec/export-xml")
async def export_xml(
    period: str,
    data_type: str,
    session: AsyncSession = Depends(get_session),
):
    """
    period: 'YYYY-MM'
    data_type: 'Акты' | 'Материалы' | 'Табель'
    """
    exporter = OneCXMLExporter(session)
    if data_type == "Акты":
        path = await exporter.export_acts_for_period(period)
    elif data_type == "Материалы":
        path = await exporter.export_materials_for_period(period)
    elif data_type == "Табель":
        path = await exporter.export_timesheet_for_period(period)
    else:
        return {"error": f"Неизвестный тип данных: {data_type}"}

    return {"path": path, "period": period, "data_type": data_type}
```

### 7.3 Подключить роутер в main.py

В `backend/app/main.py` добавить:

```python
from app.routers import integration
app.include_router(integration.router, prefix="/api/v1")
```
